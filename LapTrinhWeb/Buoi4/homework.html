Bài Tập Thực Hành DOM

Hoàn thành các yêu cầu dưới đây bằng cách viết code JavaScript và sử dụng DOM.

Yêu cầu 1: Thay đổi nội dung

Xin chào, đây là một đoạn văn bản.

Yêu cầu: Khi nhấp vào nút "Thay đổi Chào mừng", hãy thay đổi nội dung của thẻ `p` thành "Chào mừng bạn đến với bài tập DOM!".

Yêu cầu 2: Thay đổi CSS

Yêu cầu: Khi nhấp vào nút "Đổi màu Box", hãy thay đổi màu nền của thẻ `div` có ID là `box` thành màu cam.

Yêu cầu 3: Thêm và Xóa phần tử trong danh sách

  • Học JavaScript
  • Làm bài tập DOM

Yêu cầu:

  • Khi nhấp vào nút "Thêm Công việc", lấy giá trị từ ô input có ID `todoInput` và thêm một `li` mới vào danh sách `ul` có ID `todoList`. Mỗi `li` phải có nội dung là công việc nhập vào và một nút "Xóa" bên cạnh.
  • Viết hàm `removeTodo(buttonElement)` để xóa `li` khi nút "Xóa" tương ứng được nhấp. Gợi ý: có thể truy cập phần tử cha của nút để xóa nó.

Yêu cầu 4: Thay đổi thuộc tính hình ảnh

Yêu cầu: Khi nhấp vào nút "Thay đổi Hình ảnh", hãy đổi thuộc tính `src` của thẻ `img` có ID `myImage` sang một URL hình ảnh khác. Ví dụ: `https://www.svgrepo.com/show/452030/avatar-default.svg`

Yêu cầu 5: Thêm/Xóa Class CSS

Đoạn văn bản này sẽ được highlight.

Yêu cầu: Khi nhấp vào nút "Bật/Tắt Highlight", hãy thêm hoặc xóa class `highlight` (đã định nghĩa trong CSS) cho thẻ `p` có ID là `toggleClassText`. Nếu class đã có thì xóa, nếu chưa có thì thêm vào.
